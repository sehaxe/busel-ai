## Architectural Innovations

<CardGrid stencil>
  <Card title="Zero-Multiplication Inference" icon="pencil">
    Runs at 100+ tokens/s on consumer CPUs. 1.58-bit ternary quantization replaces floating-point multiplications with fast additions.
  </Card>
  <Card title="mAR Birkhoff Residuals" icon="document">
    Advanced inter-layer routing via Sinkhorn projections. Constrains layers to the Birkhoff Polytope, preventing representational collapse.
  </Card>
  <Card title="3:1 Hybrid Attention" icon="setting">
    Combines Gated DeltaNet-2 linear attention (constant-size $O(1)$ state) with Multi-Head Latent Attention (MLA) for lightweight context scaling.
  </Card>
  <Card title="Gated FastBLT Patcher" icon="star">
    Token-free, byte-level processing with a sigmoid-gated convolutional patcher, eliminating massive embedding vocabulary bloat.
  </Card>
